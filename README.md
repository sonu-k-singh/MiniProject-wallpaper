# MiniProject-wallpaper

🎵 Spotify Web Player Clone

This is a Spotify Web Player UI Clone built using HTML and CSS.
The project replicates the look and feel of Spotify's web player with a clean and modern design.

🚀 Live Demo

👉 https://mini-project-wallpaper.vercel.app

📌 Features

- 🎧 Clean and modern Spotify UI design
- 📱 Responsive layout
- 🎨 Styled using pure CSS
- 🧭 Sidebar navigation
- ▶️ Music player layout (UI only)

🛠️ Tech Stack

- HTML5
- CSS3
